## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/thyagoaraujo/gobarber-web.git

# Go into the repository
$ cd gobarber-web

# Install dependencies
$ yarn install

# Run the app
$ yarn start
```
